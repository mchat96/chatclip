# chatclip1.1

伝言テンプレート自動生成アプリ
skypeやslagなどで業務連絡をする方に。


必要項目を入力してPOWを押すと、
POWを押した時間を含めた文章が
クリップボードに保存されます。
（オプションで任意の時間にも設定可）



resultフォームで
クリップボードにコピーされた内容を
確認できます。


--------+




# 入力規則




* company			会社名を入力してください
* name				担当者名を入力してください
(会社名、担当者名のいずれかは入力必須です)

* phone number		電話番号、連絡先などを入力してください（文字入力可）
* other 			備考事項を入力してください
* time(Option)		出力する時間を設定できます。空欄の場合はPOWを押した時間が出力されます

* POWボタン			生成した文章をクリップボードにコピーし、resultエリアに表示します。

* result			出力エリアです。時刻にはPOWボタンを押した時間が出力されます
* clear 			出力エリアをクリアします。必要でしたらお使いください





タブで移動していくといい感じに使いやすそうです。
